# Shelf — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Shelf across platforms (Android, Windows, web) and establish it as the definitive smart launcher for power users across all devices.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Access shelves from Android phone
- Notification of new shared shelf items
- Quick add items from Android share sheet
- Google Play Store listing

### Windows Desktop App
- Shelf for Windows (Electron or Tauri)
- Monitor Windows app usage alongside Mac
- Unified shelf accessible from both platforms
- Shared shelf config via cloud (same iCloud or separate cloud backend)

### Web App (PWA)
- Full-featured web app at shelf.app
- Access shelves from any browser
- Read-only initially, read-write for Pro users
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app with spatial UI
- Shelf items arranged in 3D space
- Gaze + gesture navigation
- "Work mode" spatial layout: project items arranged around you

### Open Source Core
- Open-source the shelf indexing and ML core engine (shelf-core on GitHub)
- MIT license for core, proprietary license for UI
- Community can build alternative UIs, integrations, or research use

### Integration Ecosystem
- Integration registry at shelf.app/integrations
- VS Code extension: shelf integration in editor
- Figma plugin: add Figma files to shelf
- Slack/Discord bots: notify channel when shelf item added
- Zapier/Make/IFTTT: full trigger/action library

### Public Roadmap
- Public roadmap at shelf.app/roadmap (Canny or GitHub Discussions)
- User voting
- Monthly changelog posts
- Community Discord

---

## Out of Scope
- Cryptocurrency or NFT tracking
- Hardware products
- Investment/portfolio management
