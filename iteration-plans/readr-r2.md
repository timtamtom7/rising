# Readr — Iteration 2 Plan

## What was built (Iteration 1 deep pass)
- Onboarding (4 screens with SF Symbol illustrations)
- Pricing tiers (Free/Pro/Scholar)
- Error states (camera denied, OCR failed, storage full)
- App Store assets (description, keywords, real book/quote examples)
- Custom quote card designs, page texture effects

## What's still missing (Iteration 2 focus)

### 1. Organization & Discovery
- Book collections/shelves (custom shelves: "Currently Reading", "Favorites", "To Get")
- Tag system for quotes (custom tags per quote)
- Search across all quotes and books
- Sort: by book, by date added, by quote length

### 2. Export & Citations
- Export single quote as image (beautiful card format, shareable)
- Export all quotes as PDF (formatted with book, author, page number)
- Citation formats: APA, MLA, Chicago, BibTeX
- Export to Readwise (actual integration)
- Export to Notion/Obsidian

### 3. Quote Presentation
- Beautiful quote card generator (multiple design templates)
- Share to Instagram/Stories (square quote image)
- Share quote as text with attribution
- Dark mode quote cards

### 4. Book Features
- Add book manually (no photo) — just title + author
- Edit book details
- "Random quote" feature — surface a random quote from library
- Reading challenge: set a goal (X quotes per month)

### 5. OCR Refinements
- Multi-page scan (scan multiple pages of same book, link to same book)
- Better crop detection for curved pages (books spine)
- Auto-detect chapter headings
- Import from PDF

### 6. Widget
- Home screen widget: random quote of the day
- Small widget: last captured quote
- Medium widget: quote card with book title

## Custom Graphics for Iteration 2
- Quote card image templates (3-4 designs)
- Book shelf/collection visualization
- Share-to-Instagram mockup
- Widget mockups
- Tag system icons
