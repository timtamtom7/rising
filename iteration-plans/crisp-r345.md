# Crisp — Rounds 3, 4, 5

## Round 3: Sharing, Smart Features, Polish

### Share & Export
- Share transcription as PDF (formatted with title, date, transcription)
- Share as plain text
- AirDrop support
- Export all notes as ZIP/JSON

### AI Features
- AI summary of each transcription (Apple on-device AI)
- Keyword extraction
- Detect action items (things that sound like tasks)
- "Link to calendar" — if transcription mentions a date/time, offer to create a calendar event

### Widget Expansion
- Home screen widget (show last recording, quick record button)
- Small widget: last recording
- Medium widget: last 3 recordings

### Customization
- Custom waveform color
- Recording quality (high/standard)
- Auto-delete after X days (privacy setting)
- Rename recording title

---

## Round 4: Bulk, Advanced, Monetization

### Bulk Operations
- Bulk delete (select multiple, delete)
- Bulk move to folder
- Bulk export selected notes

### Advanced Editing
- Merge two recordings into one note
- Split one recording into two (at a chosen point)
- Edit transcription text (user can correct errors)

### Monetization Polish
- Lock advanced features behind paywall visually
- Trial period for Pro features (3 days free trial)
- Paywall shown after 5 recordings for Free users

### Keyboard Extension (Phase 2)
- Custom keyboard with waveform + mic button
- Long-press mic → dictation → text appears in any text field
- Gradient animation while speaking

## Round 5: Platform, Scale, Community

### Platform
- iPad native version (not scaled iPhone)
- macOS version (Catalyst or native)
- Apple Watch standalone app (record from watch)

### Sync
- iCloud sync (optional, user opts in)
- Sync across devices
- Conflict resolution for edits

### Community (Phase 2)
- Share a recording as a link (public, listen once)
