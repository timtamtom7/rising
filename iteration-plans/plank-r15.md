# Plank — R15: iOS Companion App

## Goal
Ship Plank as a native iOS app for theme browsing, preset management, and quick customization.

---

## Scope

### iOS App
- Browse theme marketplace
- Manage installed themes
- Preview themes before applying
- Apply themes to paired Mac (via iCloud)
- Preset management

### iPad Optimization
- Full iPad layout
- Sidebar + preview split
- Keyboard shortcuts

### Apple Watch App
- Glance: current theme name + accent color
- Complication: theme color swatch
- Quick preset switch

### iOS Widgets
- Small widget: current theme swatch
- Medium widget: theme + 2 presets
- Large widget: full theme browser + apply
- Lock screen widget: theme accent color
- Interactive widget: apply preset

### Watch Face Sync
- Sync theme accent color to Apple Watch face
- Dynamic: watch face updates when Mac theme changes

### Notifications
- "New theme in marketplace you might like"
- "Your theme was updated"

### Siri & Shortcuts
- "Switch to Dark Mode" → apply dark preset
- "Apply [Theme Name]" → apply specific theme

---

## Out of Scope
- Full Plank functionality on iOS (Mac is primary)
- Controlling Mac themes from watch directly
