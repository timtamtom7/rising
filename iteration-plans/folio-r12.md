# Folio — R12: Collaboration & Document Sharing

## Goal
Enable Folio for teams with shared libraries, collaborative annotation, and approval workflows.

---

## Scope

### Shared Libraries
- Share document libraries via iCloud
- Team members can view, search, download shared documents
- Real-time sync of new additions
- Role-based access: Admin, Editor, Viewer

### Collaborative Annotation
- Add comments to shared documents
- @mention teammates in comments
- Annotation threads with resolution
- "Approved" / "Rejected" status per annotation

### Document Approval
- Send document for approval
- Approver receives notification
- Approve / Request Changes / Reject workflow
- Approval history log

### Version History (Shared)
- Full version history for shared documents
- Compare versions side by side
- Restore previous version
- Who changed what and when

### Client Sharing
- Generate share link for external clients
- Read-only access with download option
- Time-limited links (7/30/90 days)
- Track when client viewed/downloaded

### Template Sharing
- Share document templates within team
- Template library accessible to all team members

---

## Out of Scope
- Real-time collaborative document editing (use dedicated apps)
- Video/audio document support
- Full DMS (Document Management System)
