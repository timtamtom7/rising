- Pricing page (Free/Homebuyer/Investor)
- Onboarding (4 screens)
- Goal setting with photo
- Deposit tracking
- Progress visualization
- Error states
- CSS variables + validation

## What's still missing (Iteration 2 focus)

### 1. Property Details
- Add multiple properties (not just one goal)
- Property address, price, link
- Photos of the property
- Notes per property

### 2. Real Market Data
- Pull estimated property value from Zillow/Realtor API
- Estimated mortgage payment calculator
- "How much do you need?" recalculation based on actual property price
- Market trends (is this neighborhood going up?)

### 3. Agent Connection
- "Find an agent" — link to Zillow/yellow pages for agent search
- Agent contact storage (name, phone, email)
- Agent notes ("Met at open house March 15")

### 4. Milestones
- Milestone tracker: "Get pre-approved → Save down payment → Make offer → Close"
- Pre-approval tracker (amount, lender, date)
- Offer tracker (offer amount, accepted/rejected, closing date)

### 5. Notifications & Reminders
- "Deposit this week?" reminder
- "Property X has been on market for 30 days — price drop likely?"
- "You're X% to your goal"

### 6. Charts & Visualization
- Monthly deposit history chart
- Goal completion projection (will you reach it by date X?)
- Comparison: your savings vs required down payment curve
