# Fume — Rounds 3, 4, 5

## Round 3: Cross-Notes, Import/Export, Polish

### Source Connections
- "Related notes" section on each source
- Keyword overlap detection
- Visual connection lines between related sources
- "People who added this also added..."

### Import Expansion
- Import from Obsidian (folder sync)
- Import from Notion (markdown export)
- Import from Apple Notes
- Import from Pinboard, Pocket, Instapaper

### Export Expansion
- Export to Obsidian (markdown files)
- Export to Notion (via API or markdown)
- Export conversation history as PDF
- Export all sources as JSON

### Search Polish
- Advanced query syntax ("from:March about X")
- Search within a specific collection/tag
- Search by date range
- Semantic search improvement

## Round 4: AI, Intelligence, Automation

### AI Analysis (Real)
- Connect to local LLM (Apple's on-device or Ollama)
- "Summarize this note in 3 sentences"
- "What are the key action items?"
- "What questions does this raise?"

### Automatic Intelligence
- Auto-tag sources based on content
- Auto-generate summary for each source
- Key facts extraction (dates, names, numbers mentioned)
- "What changed since last week?" — diff for updated notes

### Automation
- "If I add a source with X tag, automatically add tag Y"
- Daily digest: "You added X new sources this week"
- Weekly review prompt

### Widget
- Home screen widget: quick query — ask Fume a question from widget
- Small: last source added
- Medium: last 3 sources
