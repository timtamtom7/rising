# Kale — Iteration 2 Plan

## What was built (Iteration 1 deep pass)
- Onboarding (4 screens)
- Pricing tiers (Free/Daily/Complete)
- Barcode scanning, tap-to-confirm cards
- Monthly heatmap calendar
- Error states
- App Store assets

## What's still missing (Iteration 2 focus)

### 1. Smart Features
- **Interaction detection**: detect if vitamins interact badly (basic check — e.g., calcium + iron compete for absorption)
- **Dosage reminders**: "You took Vitamin D but no Magnesium — consider taking them together"
- **Low stock alerts**: when a vitamin is running low based on daily usage
- **Monthly consistency report**: "You were 90% consistent in March. February was 75%."

### 2. Multi-user / Family
- Complete tier: family sharing (up to 6 members)
- Each member has their own vitamins and tracking
- Family consistency competitions
- "Who in the family is most consistent?"

### 3. Calendar Integration
- Sync with Apple Calendar
- Show vitamin reminders as calendar events
- Import calendar events to predict days you'll be traveling (skip reminders)

### 4. Better History
- Per-vitamin history (when did you last take it?)
- Per-day detail view
- "Best streak" tracking per vitamin
- Streak recovery: "You missed 3 days — re-start your streak?"

### 5. Supplement Details
- Tap a vitamin → info sheet: what it does, recommended timing, food sources
- "Take with food" warnings
- "Best taken with Vitamin C for absorption" hints

### 6. Widget
- Home screen widget: today's vitamins + one-tap check
- Small widget: one-tap record
- Medium widget: all today's vitamins with check buttons

## Custom Graphics for Iteration 2
- Streak badge/medal icons
- Consistency report card
- Low stock alert visual
- Widget mockups
