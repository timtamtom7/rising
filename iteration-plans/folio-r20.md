# Folio — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Folio across platforms and establish it as the definitive document management app for personal and professional use.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full document capture and management
- Google Play Store listing

### Windows / Linux Desktop
- Folio for Windows/Linux (Electron or Tauri)
- Same features as macOS app
- Windows Store and Linux packages

### Web App (PWA)
- Document access at folio.app
- View, search, download
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial document viewer
- 3D document organization
- Immersive reading mode

### Open Source Core
- Open-source the document processing engine (folio-core on GitHub)
- MIT license
- Community can build alternative UIs

### Integration Ecosystem
- Integration registry at folio.app/integrations
- QuickBooks, Xero: receipt export
- Notion, Obsidian: document link
- Zapier/Make/IFTTT: document triggers
- Evernote, Notion import

### Public Roadmap
- Public roadmap at folio.app/roadmap
- User voting on features
- Monthly blog posts

---

## Out of Scope
- Full ECM features
- Video document support
- Automated document workflows
