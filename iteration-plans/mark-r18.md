# Mark — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Mark widgets across macOS and iOS for quick bookmark access without opening the app.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: recent bookmark
- Medium widget: 3 recent bookmarks + quick search
- Large widget: full library + collections + search + add
- Interactive widget: add bookmark directly from widget
- Widget stack: recent + collections

### iOS Home Screen Widgets
- Small: recent bookmark
- Medium: recent + quick access
- Large: full dashboard
- Lock screen widget: recent bookmark
- Interactive widget: add bookmark

### Reading List Widget
- "Read Later" queue
- Estimated reading time
- Progress: read / unread

### Collection Widget
- Specific collection
- Recent additions
- Item count

### Widget Configuration
- In-app widget setup
- Choose which collections to show
- Select default view

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity
