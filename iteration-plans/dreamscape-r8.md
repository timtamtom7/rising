# Dreamscape — Round 8: iPad App, macOS App, Cross-Platform

## Features

### iPad App
- Native iPad layout — side-by-side dream map and dream detail
- Apple Pencil support — draw your dream, AI interprets the drawing
- Large screen dream gallery — magazine-style layout

### macOS App
- Menu bar app — quick dream entry without opening app
- Widget for macOS dashboard
- Native macOS design — matches Apple's productivity apps

### Continuity
- Handoff — start dream on iPhone, continue on iPad
- Universal clipboard — copy dream text anywhere
- iCloud sync already built in R2 — works across all devices

## Design Polish
- iPad: multi-column layout, split view support
- macOS: native menu bar, SF Pro, minimal window chrome
- Cross-platform design language consistent
