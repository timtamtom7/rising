# COIN — R9: Beta Program, TestFlight, Feedback

## Goal
External beta testing: TestFlight, feedback, iteration.

---

## Scope

### TestFlight Setup
- App Store Connect beta app
- External testers: public link or invite
- Internal testers
- Build notes per release

### CI/CD for Beta
- Auto-build on main push
- Auto-upload to App Store Connect
- Version: `X.Y.Z-beta.N`

### Feedback System
- In-app "Send Feedback"
- Structured feedback form
- Feature request voting
- Email fallback

### Bug Reporting
- Template: steps to reproduce, expected vs actual
- Screenshot/video attachment
- System info included

### Beta Communication
- In-app "What's New in Beta"
- Changelog per build
- Tester announcements
- Discord/Slack (optional)

### Beta Metrics
- Feature flags for beta features
- Usage tracking (opt-in)
- Crash monitoring

### TestFlight Review
- Triage feedback
- Critical fixes first
- Iterate on UX issues
- Stabilize for 1.0

---

## Out of Scope (R10)
- Public launch
- Marketing
