# Grapple — Round 2: Debate Modes, Sources, Synthesis History

## Context
R1 built: text input, counter-arguments, rebuttal, synthesis, dark slate debate court. R2: debate modes, source integration, deeper synthesis.

## Features

### Debate Modes
- **Quick** — 3 quick counter-arguments, no rebuttal round
- **Standard** — the full current flow (5 arguments, rebuttals, synthesis)
- **Deep Dive** — multiple rounds, AI asks follow-up questions, 20+ arguments
- **Opposing View** — instead of counter-arguments, AI takes the opposing side fully and argues it

### Source Integration
- Paste a link, article, or document — Grapple reads it and uses it as evidence
- Web search integration — AI searches for factual counter-data
- Citation system — counter-arguments reference sources
- "Is this claim factually accurate?" check

### Synthesis History
- Past synthesis sessions — searchable, grouped by topic
- Synthesis evolution — "Last time you grappled X, you concluded Y. Here's what changed."
- Export synthesis as PDF or shareable link
- Topic tracking — what topics do you grapple most?

### Real-Time Fact Check
- After synthesis, AI highlights: "The claim 'X' — actual data suggests Y"
- Confidence indicator on each point
- "Change my mind" — if new evidence appears, you can reopen a session

### Design Polish
- Arguments snap in with impact — slight bounce
- Color coding by strength (weak=dim red, strong=bright red)
- Synthesis card — appears with dramatic reveal animation
- SF Mono throughout for academic precision
