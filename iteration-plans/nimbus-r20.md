# Nimbus — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Nimbus to all major platforms and establish it as the definitive personal and team knowledge management tool.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full note-taking experience
- Material You dynamic color
- Google Play Store listing

### Windows / Linux Desktop
- Nimbus for Windows/Linux (Electron or Tauri)
- Same rich text editor and features
- Windows Store and Linux package distributions

### Web App (PWA)
- Full-featured web app at nimbus.app
- Rich text editor in browser
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial notes: arrange notes in 3D space
- 3D knowledge graph visualization
- Immersive reading mode

### Open Source Core
- Open-source the note storage and sync engine (nimbus-core on GitHub)
- MIT license
- Community can build alternative UIs
- Researchers can study knowledge management patterns

### Integration Ecosystem
- Integration registry at nimbus.app/integrations
- Slack: note links shared in Slack show preview
- Jira/Linear: link tasks to notes
- Raycast/OLauncher: quick note capture
- Zapier/Make/IFTTT: note create/update triggers

### Public Roadmap
- Public roadmap at nimbus.app/roadmap
- User voting on features
- Monthly blog posts

---

## Out of Scope
- Full project management features
- Time tracking
- Video conferencing
