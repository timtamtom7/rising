# Cast — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Cast across platforms and establish it as the definitive casting and streaming management tool.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full casting control
- Google Cast SDK integration
- Google Play Store listing

### Windows / Linux Desktop
- Cast for Windows/Linux (Electron or Tauri)
- Same features as macOS app
- Windows Cast SDK support
- Windows Store and Linux packages

### Web App (PWA)
- Remote casting control at cast.app
- Watch party access from browser
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial casting dashboard
- 3D queue visualization
- Casting control from visionOS

### Open Source Core
- Open-source the casting discovery engine (cast-core on GitHub)
- MIT license
- Community can build custom receivers

### Integration Ecosystem
- Integration registry at cast.app/integrations
- Smart TV apps: Samsung, LG, Vizio
- Streaming service APIs: YouTube, Twitch, custom streams
- Home automation: Control4, Savant
- Zapier/Make/IFTTT: casting triggers

### Public Roadmap
- Public roadmap at cast.app/roadmap
- User voting on features
- Monthly blog posts

---

## Out of Scope
- Pirated content aggregation
- Subscription service aggregation
- Content rights management
