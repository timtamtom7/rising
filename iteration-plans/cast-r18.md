# Cast — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Cast widgets across macOS and iOS for quick casting control.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: now playing
- Medium widget: now playing + device + play/pause
- Large widget: full control + queue + devices
- Interactive widget: play/pause/stop directly
- Widget stack: now playing + queue

### iOS Home Screen Widgets
- Small: now playing
- Medium: now playing + queue
- Large: full dashboard
- Lock screen widget: now playing
- Interactive widget: transport controls

### Device Widget
- Quick device selector
- One-tap cast to favorite device

### Queue Widget
- Current queue
- Next up
- Add to queue

### Widget Configuration
- In-app widget setup
- Choose favorite device
- Select default view

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity for casting
