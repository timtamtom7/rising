# Perch — Iteration 2 Plan

## What was built (Iteration 1 deep pass)
- Onboarding (4 screens)
- Pricing tiers (Free/Wander/Explorer)
- Passive location tracking (CLLocationManager)
- Trip creation + MapKit visualization
- CO₂ estimation
- Error states

## What's still missing (Iteration 2 focus)

### 1. Trip Templates
- Save a trip as a template ("Weekend in Paris" template)
- Re-use templates for recurring trips
- Templates auto-fill destinations you've visited before

### 2. Better Location Detection
- Automatic trip detection (user is traveling → Perch asks "Are you on a trip?")
- Auto-end trip when user returns home (detect home location)
- Better city detection (more accurate geocoding)

### 3. Travel Insights
- Countries/cities ranking (most visited)
- Total distance traveled
- CO₂ total for the year
- "Travel frequency" — how often do you travel?
- Travel streaks (consecutive months with travel)

### 4. Itinerary Export
- Export trip as PDF itinerary
- Share trip summary with family/friends
- Export to Apple Maps / Google Maps

### 5. Multi-trip Management
- Run multiple trips simultaneously (for digital nomads)
- Trip comparison ("This trip was 3 days longer than your average")

### 6. Privacy
- "Private trip" — don't show in travel history
- Trip diary: add notes to specific cities in a trip

## Custom Graphics for Iteration 2
- Trip template card
- World map with visited countries highlighted
- CO₂ footprint comparison graphic
- Travel frequency chart
