# Caliber — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Caliber's presence across macOS and iOS with widgets for study motivation and quick review access.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: due cards count + streak
- Medium widget: 7-day study chart + due count + next review time
- Large widget: full dashboard — streak, retention rate, due cards, weekly goal progress
- Interactive widget: "Start Review" launches directly
- Widget stack: combine due count + streak + insight widgets

### iOS Home Screen Widgets
- Small: due cards count
- Medium: streak + due count + retention rate
- Large: study dashboard with chart
- Lock screen widget: streak count
- Interactive widget: start review from widget

### iOS Lock Screen Widgets
- Circular: streak as ring
- Rectangular: streak + due today
- Inline: "📚 X cards due"

### Study Timer Widget
- Large widget: Pomodoro study timer
- "Session Complete!" notification when timer ends
- Auto-start review session after focus block

### Study Streak Visualization
- Year calendar widget: days studied in color
- Heat map widget: study intensity over 30 days
- Achievement badges for milestones (7-day, 30-day, 100-day streak)

### Widget Configuration
- In-app widget configuration gallery
- Choose which deck(s) to show
- Choose metric to display prominently

---

## Out of Scope
- Android widgets
- Windows desktop widgets
- Live Activity (future)
