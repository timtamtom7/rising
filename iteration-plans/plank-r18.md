# Plank — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Plank widgets across macOS and iOS for quick theme access and preview.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: current theme accent color swatch
- Medium widget: current theme + 2 quick presets
- Large widget: theme browser + preset list
- Interactive widget: apply preset directly from widget
- Widget stack: accent + presets

### iOS Home Screen Widgets
- Small: theme accent color
- Medium: theme + presets
- Large: full theme browser
- Lock screen widget: accent color ring
- Interactive widget: apply preset

### Widget Configuration
- In-app widget setup
- Choose which presets to show
- Select theme for accent widget

### Desktop Widget (macOS 14+)
- Desktop widget showing current theme
- Tap to open Plank

### Notification Center Widget
- Theme quick actions from Notification Center
- Recent themes

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity
