# Bridge — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Bridge widgets across macOS and iOS for quick device control without opening the app.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: device count
- Medium widget: 4 favorite devices + connect/disconnect
- Large widget: full device dashboard + scenes
- Interactive widget: connect/disconnect device directly
- Widget stack: devices + scenes

### iOS Home Screen Widgets
- Small: device count
- Medium: device list + quick actions
- Large: dashboard + scenes + workflows
- Lock screen widget: scene buttons
- Interactive widget: activate scene

### Scene Widgets
- Dedicated widget for scene activation
- "Movie Night", "Work", "Sleep" scene buttons
- Tap to activate directly

### Device Status Widget
- Real-time device status (connected/disconnected)
- Battery level for battery-powered devices
- Network signal strength

### Widget Configuration
- In-app widget setup
- Choose which devices/scenes to show
- Favorite devices for quick access

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity for device events
