# Zones — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Zones across platforms and establish it as the definitive context-aware zone management tool.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full zone management
- Material You dynamic color
- Google Play Store listing

### Windows / Linux Desktop
- Zones for Windows/Linux (Electron or Tauri)
- Same features as macOS app
- Windows Store and Linux packages

### Web App (PWA)
- Zone management at zones.app
- View zones and history from browser
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial zone visualization
- 3D zone dashboard
- Mac control from visionOS

### Open Source Core
- Open-source the zone detection engine (zones-core on GitHub)
- MIT license
- Community can build alternative UIs

### Integration Ecosystem
- Integration registry at zones.app/integrations
- HomeKit: full scene control
- Philips Hue, LIFX: zone lighting
- Sonos: zone-based audio
- Zapier/Make/IFTTT: zone triggers

### Public Roadmap
- Public roadmap at zones.app/roadmap
- User voting on features
- Monthly blog posts

---

## Out of Scope
- Social location sharing
- Dating/meetup features
- GPS tracking history
