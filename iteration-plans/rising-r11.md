# Rising — Round 11: Zillow Integration, Milestones, Notifications

## Goal
Complete Rising's home buying toolkit with Zillow integration, detailed milestones, and smart notifications.

---

## Scope

### Zillow Integration
- Property search via Zillow API
- "Save from Zillow" deep link into Rising
- Price history charts from Zillow data
- "For sale near this property" suggestions
- Zestimate comparison to target purchase price
- School district ratings overlay

### Milestone Tracking
- "Down payment saved" celebration animation
- "Pre-approval received" milestone
- "First home tour completed" milestone
- "Offer submitted" with tracking
- "Home inspection passed" milestone
- Custom milestones user-defined
- Milestone timeline view

### Smart Notifications
- Price drop alerts for saved properties
- "Your saved home is back on market"
- "Mortgage rates changed" alerts
- "You can afford X% more home than last month" (when rates drop)
- "Property X has been on market 30+ days — price drop likely?"
- Monthly progress digest

### Calculator Enhancements
- Rent vs buy comparison calculator
- Affordability calculator with all debt factored
- Investment return projections if buying vs renting
- "What if I put 10% down vs 20% down" comparison
- Property tax and HOA cost estimator

---

## Out of Scope
- Agent chat integration (R12)
- Mortgage pre-approval flow (R13)
