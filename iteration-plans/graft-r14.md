# graft — Round 14: Android App, Web Platform, Cross-Platform Sync

## Goal
Expand Graft to Android and web — ensuring skill tracking is accessible on any device and enabling cross-platform sync so practice sessions are captured anywhere.

---

## Scope

### Android App
- Full Graft app on Android (Google Play Store)
- Feature parity: practice logging, skill tracking, AI coaching, challenges
- Material Design 3 UI
- Android widgets: quick-log practice session widget, daily challenge widget
- Wear OS companion: start/stop practice sessions from watch

### Web Platform
- Responsive web app at graft.app
- View practice history, skill levels, challenges, leaderboards
- Log practice sessions from browser
- Web-based teacher dashboard for mentors
- Browser notifications for reminders, challenges

### Cross-Platform Sync
- Unified backend (Firebase or custom)
- All practice data syncs across iOS + Android + web
- Real-time leaderboard updates
- Offline support: log practice offline, sync when online

### Skill-Specific Apps (Mini)
- **Graft Guitar** (separate app): Deep guitar integration — metronome, tuner, chord library, song learning
- **Graft Chess** (separate app): Chess-specific — track puzzles, games, opening preparation
- **Graft Language** (separate app): Language learning integration — Duolingo sync, conversation practice logging
- Each mini-app links to main Graft profile

---

## Out of Scope
- Subscription business optimization (R15)
- Graft 2.0 redesign (R16)
