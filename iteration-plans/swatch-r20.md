# Swatch — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Swatch across all major platforms and establish it as the definitive color management tool for designers and teams.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full palette management, color picker
- Material You dynamic color support
- Sync with iOS/macOS via cloud
- Google Play Store listing

### Web App (PWA)
- Full-featured web app at swatch.app
- Palette management from browser
- Color picker from webcam
- Export in all formats
- Offline support via Service Worker

### Windows / Linux Desktop
- Swatch for Windows/Linux (Electron or Tauri)
- Same feature set as macOS app
- Windows Store listing
- Linux packages (.deb, .rpm, .AppImage)

### Apple Vision Pro
- visionOS app
- Spatial color palette: arrange colors in 3D space
- 3D color visualization
- Spatial design presentation mode

### Open Source Core
- Open-source the color extraction and ML engine (swatch-core on GitHub)
- MIT license
- Community can build alternative UIs or integrations

### Integration Ecosystem
- Integration registry at swatch.app/integrations
- Adobe Creative Cloud integration (Photoshop, Illustrator, InDesign)
- Figma full双向 sync
- Microsoft Office color palette export
- AutoCAD / SolidWorks color export
- Zapier/Make/IFTTT: palette change triggers

### Community
- Public palette gallery at swatch.app/explore
- Upload and share palettes publicly
- Palette of the week feature
- Community color challenges

---

## Out of Scope
- Hardware color calibration tools
- Print production workflow (RIP,分离)
- Display calibration
