# Volt — R13: Enterprise Features & MDM Integration

## Goal
Deepen enterprise capabilities with MDM support, fleet analytics, compliance reporting, and volume licensing.

---

## Scope

### MDM Deeper Integration
- Full ManagedAppleDevice support via Apple Business Manager
- Configuration Profiles pushed from MDM server
- Remote lock/wipe of Volt data on managed devices (if device is lost)
- Device enrollment check: show IT whether Mac is enrolled in MDM

### Compliance & Audit Reporting
- Generate battery health compliance reports per device or fleet
- SOC 2 Type II readiness documentation (available on request)
- GDPR/CCPA compliance documentation for enterprise
- Export audit log: who changed what power settings and when

### Volume Purchase Program (VPP)
- Enroll in Apple VPP for volume distribution
- Redemption codes or direct push to managed devices
- License usage dashboard for IT administrators

### Enterprise Dashboard
- Web-based admin console at enterprise.volt.app
- View all enrolled devices, battery health, power mode status
- Set fleet-wide power policies
- Generate monthly/quarterly energy consumption reports
- Budget alerts when fleet power consumption exceeds thresholds

### SSO / SAML 2.0
- Enterprise SSO for dashboard access
- Support for Okta, Azure AD, Google Workspace
- Role-based access control (Admin, Read-Only, User)

---

## Out of Scope
- Hardware asset management beyond battery
- Integration with ITSM tools (ServiceNow, Jira)
- Custom branding / white-label
