# Readr — Rounds 3, 4, 5

## Round 3: Social, Discovery, Polish

### Quote Sharing
- Share quote as beautiful image (multiple templates)
- Share to Instagram/Stories (square quote image)
- Share quote as text with attribution
- Dark mode quote cards

### Reading Lists Expansion
- "Want to Read" list (before you start)
- Custom shelves: user-created
- Mark books as abandoned (didn't finish)
- Sort books by: date added, title, author, quote count

### Random Quote Feature
- "Random quote" — surface a random quote from library
- Daily quote notification (configurable)

### Tags
- Add custom tags per quote
- Filter quotes by tag
- Tag management view

## Round 4: Recommendations, Notes, Export

### Book Recommendations
- Based on genres you read (detect from book titles)
- "Readers who liked X also read Y"
- Curated lists (manual or API)

### Notes & Highlights
- Add personal notes to specific books
- Highlight specific passages within a quote
- Link notes to specific pages

### Advanced Export
- Export all quotes as PDF book (formatted)
- Export to Readwise (real API integration)
- Export to Notion/Obsidian
- Citation formats: APA, MLA, Chicago, BibTeX

### OCR Refinements
- Multi-page scan (scan multiple pages of same book, link to same book)
- Better crop detection for curved pages (book spine)
- Import from PDF
