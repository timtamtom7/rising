# Notch — R12: Collaboration & Team Configurations

## Goal
Enable Notch for teams with shared menu bar configurations and IT-managed deployments.

---

## Scope

### Shared Configurations
- Share menu bar configurations via iCloud
- Team can share "Developer Setup", "Design Setup", etc.
- Real-time sync of shared configs
- Import/export configs as files

### Team Presets Library
- Publish presets to team library
- Browse and apply team presets
- Version control for shared presets

### IT-Managed Deployments
- MDM support for enterprise Mac deployment
- IT can push menu bar configurations to managed Macs
- Lock certain menu bar items (can't be removed)
- Audit which machines have which configs

### Role-Based Access
- Admin can manage team presets
- Member can apply presets, contribute suggestions
- Viewer can browse and download

### Guest Access
- Invite external consultant to review menu bar setup
- Read-only access to team presets
- Time-limited access

---

## Out of Scope
- Real-time collaborative menu bar editing
- Automatic installation of apps mentioned in presets
