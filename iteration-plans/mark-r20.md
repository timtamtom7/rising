# Mark — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Mark across platforms and establish it as the definitive bookmark and knowledge management tool.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full bookmark management
- Chrome extension sync
- Google Play Store listing

### Windows / Linux Desktop
- Mark for Windows/Linux (Electron or Tauri)
- Browser extensions for Chrome, Firefox, Edge
- Same features as macOS app
- Windows Store and Linux packages

### Web App (PWA)
- Bookmark access at mark.app
- Full bookmark management from browser
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial bookmark organization
- 3D collection visualization
- Immersive reading mode

### Open Source Core
- Open-source the bookmark engine (mark-core on GitHub)
- MIT license
- Community can build alternative UIs

### Browser Extensions
- Safari, Chrome, Firefox, Edge extensions
- Full feature parity across browsers
- One-click bookmark, right-click menu, keyboard shortcuts

### Integration Ecosystem
- Integration registry at mark.app/integrations
- Read-it-later apps: Pocket, Instapaper import
- Note apps: Notion, Obsidian link
- RSS readers: Feedly, Reeder import
- Zapier/Make/IFTTT: bookmark triggers

### Public Roadmap
- Public roadmap at mark.app/roadmap
- User voting on features
- Monthly blog posts

---

## Out of Scope
- Full social network
- Content creation beyond bookmarks
- Video/book/audio bookmarking
