## What was built (Iteration 1 deep pass)
- Pricing page (Free/Pro/Teams)
- Onboarding (4 screens)
- Error states

- Critical save/navigation bug fix
- Full flow tested

## What's still missing (Iteration 2 focus)

### 1. AI Decision Advice
- When user has listed options, AI analyzes and suggests: "Have you considered...?"
- Risk assessment for each option
- "Worst case / best case" modeling (already partially in spec but needs implementation)
- Decision confidence score based on how complete the framework is

### 2. Decision Reminders & Check-ins
- "Your decision deadline is in 2 days — have you decided yet?"
- Weekly nudge for active, undated decisions
- "You made this decision X days ago — how did it go?"
- Post-decision reflection (outcome tracking)

### 3. History & Patterns
- Decision history with outcomes
- "Decisions made" count
- "Decisions regretted" tracking (optional — let user mark as good/bad outcome)
- Pattern detection: "You make big decisions faster when you sleep on it"

### 4. Collaboration
- Teams tier: invite collaborator to a decision
- Shared decision view (both see same page)
- Comments on specific options

### 5. Export
- Export decision as PDF
- Share decision framework as a link
- Print-friendly version

### 6. Better Decision Framework
- Add "stake" — how much does this decision matter?
- Add "reversibility" — can you easily undo this?
- Add "time horizon" — short-term vs long-term impact
- Visual weight: some decisions are bigger than others
