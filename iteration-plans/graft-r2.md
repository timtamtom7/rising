# Graft — Iteration 2 Plan

## What was built (Iteration 1 deep pass)
- Onboarding (4 screens)
- Pricing tiers (Free/Track/Master)
- Single skill tracking (weekly dots, monthly heatmap)
- Session logging with duration + feel rating
- Skill picker (20 presets + custom)
- Error states

## What's still missing (Iteration 2 focus)

### 1. Multiple Skills (Master tier)
- Track up to 5 skills simultaneously (Master tier)
- Skill switching in the app
- Per-skill stats and history
- Compare skills: "You practiced guitar 2h vs piano 1h this week"

### 2. Analytics & Insights
- Practice trend line (are you practicing more or less?)
- "You're in a plateau" — if hours haven't changed in 2 weeks
- Personal records: longest session, best feel rating, most consistent week
- Goal setting: "Practice 5 hours this month" — progress tracking

### 3. Practice Plan / Structure
- Planned practice sessions ("Practice guitar Tuesday 7pm")
- Reminder for planned sessions
- Distinguish "planned" vs "actual" practice

### 4. Data Export
- Export practice history as CSV
- Generate a "practice report" for teachers/coaches
- Share progress as image (for sharing on social)

### 5. Timer Mode
- Built-in practice timer (in-app, not just manual logging)
- Timer + feel rating at end
- Distinguish manual log vs timer-based session

### 6. Streak Mode (Optional)
- "Streak mode" — optional gamification layer
- If you miss a day, the streak resets (user opts in)
- Not forced — only for people who want the gamification

## Custom Graphics for Iteration 2
- Practice trend line chart
- Skill comparison bar chart
- Practice goal progress visual
- Timer mode UI
- Practice report template
