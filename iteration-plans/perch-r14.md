# perch — Round 14: Android App, Web Platform, Cross-Platform Sync

## Goal
Expand Perch to Android and web — making carbon-aware travel tracking accessible on any device, with full cross-platform sync.

---

## Scope

### Android App
- Full Perch app on Android (Google Play Store)
- Feature parity: trip logging, CO₂ tracking, AI insights, carbon offset purchasing
- Material Design 3 UI
- Android widgets: active trip widget, carbon budget tracker widget
- Google Maps integration: trip route visualization

### Web Platform
- Responsive web app at perch.earth
- View all trips, carbon history, CO₂ dashboard
- Browser-based trip logging
- Trip planner: plan future trips with CO₂ estimates
- Carbon offset purchasing from web

### Cross-Platform Sync
- Unified backend (Firebase or custom)
- All trip data, carbon budgets, offsets sync across iOS + Android + web
- Offline support: log trips offline, sync when online

### Passive Location Tracking
- Background location tracking for automatic trip detection
- When Perch detects you're traveling (movement patterns), prompt: "Are you on a trip?"
- Auto-categorize transport mode from movement data
- Works on iOS and Android
