# Sash — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Sash to Android, Windows, and Linux to become the definitive cross-platform sync solution with zero-knowledge security.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full sync management on Android
- Material You dynamic color
- Sync status monitoring, conflict resolution
- Google Play Store listing

### Windows Desktop
- Sash for Windows (Electron or Tauri)
- Same zero-knowledge sync protocol
- Windows Explorer integration (context menu)
- Windows Store listing

### Linux Desktop
- Sash for Linux (native or Electron)
- Desktop integration for major distros
- Package distributions: .deb, .rpm, .AppImage

### Unified Cross-Platform Protocol
- Same encrypted sync protocol across all platforms
- Seamless handoff: start on Mac, continue on Windows
- Consistent UI across platforms

### Apple Vision Pro
- visionOS app
- Spatial file browser for synced folders
- 3D sync activity visualization

### Open Source Protocol
- Open-source the sync protocol (sash-protocol on GitHub)
- MIT license
- Community can build alternative clients
- Protocol documentation for researchers

### Integration Ecosystem
- Integration registry at sash.app/integrations
- NAS support: sync with Synology, QNAP, etc.
- Home automation: trigger sync when home network detected
- Zapier/Make/IFTTT: sync triggers

### Public Roadmap
- Public roadmap at sash.app/roadmap
- User voting on features
- Monthly development updates

---

## Out of Scope
- Cloud storage provider (builds on existing providers)
- CDN or edge hosting
- Email/chat integration within Sash
