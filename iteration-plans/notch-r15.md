# Notch — R15: iOS Companion App

## Goal
Ship Notch as a native iOS app for menu bar configuration on the go and Apple Watch quick controls.

---

## Scope

### iOS App
- Menu bar configuration from iPhone and iPad
- Browse and apply presets
- See current menu bar status
- Configuration analytics
- Push notifications for configuration changes

### iPad Optimization
- Full iPad layout
- Split view: preset list + configuration
- Keyboard shortcuts
- External display for configuration overview

### Apple Watch App
- Watch: current menu bar status
- Quick show/hide for favorite items
- Complication: quick toggle
- Haptic feedback for changes

### iOS Widgets
- Small widget: current config name
- Medium widget: config + quick toggles
- Large widget: full config + presets + analytics
- Lock screen widget: current preset
- Interactive widget: toggle items

### Siri & Shortcuts
- "Show my menu bar" → apply show preset
- "Hide menu bar items" → apply hide preset
- "Toggle Wi-Fi menu"
- Shortcuts: show/hide items, apply preset

---

## Out of Scope
- Android companion (separate round)
- Real-time menu bar preview on iOS
