# DUST — R10: Public Launch

## Goal
Ship DUST 1.0 to the public App Store. Complete all pre-launch checklist items.

---

## Scope

### Final App Store Submission
- App Store Connect: complete all metadata
- Upload final build
- Select territories
- Set release date: manual or automatic
- Age rating questionnaire finalization

### Pre-Launch Checklist
- [ ] All R1-R9 out-of-scope items addressed or documented
- [ ] Privacy policy URL live
- [ ] EULA finalized
- [ ] Support URL configured
- [ ] Marketing assets uploaded
- [ ] Beta feedback incorporated
- [ ] Crash-free for 48 hours
- [ ] Performance acceptable on minimum supported hardware

### Marketing Assets
- App icon: 1024×1024 + all sizes
- Screenshots: 6 per size (macOS 12+, macOS 13+)
- App preview video
- Press kit (optional)

### Launch Day
- Monitor App Store Connect for review status
- Prepare social posts (if applicable)
- Prepare customer support responses
- Watch for crash spikes

### Post-Launch
- Respond to first reviews
- Monitor crash reporting
- Address critical issues quickly
- Plan 1.0.1 patch for any blocker bugs

### App Store Optimization
- Keyword research for "duplicate finder", "cleanup", "disk space"
- Description: lead with value proposition
- Screenshots: show before/after or key feature

### Metrics to Watch
- Download count
- Active users
- Crash rate
- Review scores
- Conversion rate (free → premium)

---

## Launch Complete
DUST 1.0 is live. Ongoing: bug fixes, feature additions per roadmap.
