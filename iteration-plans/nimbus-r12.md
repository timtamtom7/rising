# Nimbus — R12: Collaboration & Team Notes

## Goal
Enable Nimbus for teams with shared notebooks, collaborative editing, and comment workflows.

---

## Scope

### Shared Notebooks
- Share notebooks via iCloud with team members
- Real-time collaborative editing (multiple people in same note)
- Presence indicators: see who's viewing and editing
- Per-notebook permissions: Admin, Editor, Viewer

### Collaborative Features
- Inline comments: select text → add comment
- @mention teammates in comments
- Comment threads with resolution status
- Suggested edits: propose change without modifying original (tracked changes)

### Team Workspace
- Team dashboard: all shared notebooks in one place
- Activity feed: recent changes across team notebooks
- Version history for shared notebooks
- "Last edited by" attribution on every paragraph

### Guest Access
- Invite external collaborators (clients, contractors) to specific notebooks
- Read-only or contributor access
- Time-limited access links (7/30/90 days)
- External guests don't need an account

### Review Workflows
- Request review on a note
- Review status: Draft, In Review, Approved, Published
- Approval signatures (like a sign-off)
- Version locking: lock note during review

### Note Templates (Shared)
- Team templates: shared templates for meeting notes, project specs, etc.
- Template versioning: update template → existing notes don't break

---

## Out of Scope
- Video conferencing within notes
- Real-time document co-authoring with conflict resolution
- Full project management (use Linear, Notion)
