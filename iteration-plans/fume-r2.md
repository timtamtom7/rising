# Fume — Iteration 2 Plan

## What was built (Iteration 1 deep pass)
- Onboarding (4 screens)
- Pricing tiers (Free/Pro/Archive)
- Error states (AI query failed, source save failed, transcription failed, etc.)
- App Store assets
- Library grid view with sources

## What's still missing (Iteration 2 focus)

### 1. Deeper AI / Smart Features
- Source connections — AI shows which notes relate to each other
- "Related notes" section on each source
- Automatic tagging/categorization of sources
- Fact extraction from notes (key facts, dates, names)
- "What changed since last week?" — diff summary for updated notes

### 2. Better Search & Organization
- Tag system for sources
- Custom collections/folders
- Sort: by date added, date updated, most referenced
- Advanced query syntax ("show me notes from March about X")
- Search within a specific source type only

### 3. Import / Export
- Import from Obsidian (folder sync)
- Import from Notion (API or markdown export)
- Import from Apple Notes
- Export to Markdown (for Obsidian/Notion sync)
- Export conversation history

### 4. Source Types
- PDF import (not just article URLs)
- Voice memo → transcribed + indexed
- Screenshot → OCR + indexed
- Apple Books highlights import

### 5. Widget
- Home screen widget: quick query — ask Fume a question from widget
- Small: last source added
- Medium: last 3 sources

### 6. Collaboration (Phase 2)
- Share a source collection with someone
- Shared knowledge base with read-only or collaborative access

## Custom Graphics for Iteration 2
- Source connection visualization (nodes and lines)
- Widget mockups
- Collection/folder icons
- Import source type icons
