# Folio — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Folio widgets across macOS and iOS for quick document capture and access.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: recent document thumbnail
- Medium widget: 3 recent documents + quick scan button
- Large widget: full library browser + recent + search
- Interactive widget: scan document directly from widget
- Widget stack: recent + scan

### iOS Home Screen Widgets
- Small: recent document
- Medium: 3 recent + search
- Large: full library
- Lock screen widget: recent document
- Interactive widget: scan

### Quick Capture Widget
- Dedicated scan widget
- Tap → camera opens for scan
- Multi-page capture from widget

### Document Type Widget
- Filter widget by type (receipts, contracts, etc.)
- Show count per type

### Widget Gallery
- In-app widget configuration
- Preview all sizes
- Quick-add

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity
