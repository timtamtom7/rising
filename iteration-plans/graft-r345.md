# Graft — Rounds 3, 4, 5

## Round 3: Multi-Skill, Timer, Polish

### Multiple Skills (Master Tier)
- Track up to 5 skills simultaneously
- Skill switching in app
- Per-skill stats: total hours, session count, average feel
- Compare skills: "You practiced guitar 2h vs piano 1h this week"

### Built-in Timer
- Practice timer (in-app)
- Timer presets per skill type
- Timer + feel rating at end
- Distinguish manual log vs timer-based session

### Practice Plan
- Planned practice sessions ("Practice guitar Tuesday 7pm")
- Reminder for planned sessions
- Distinguish "planned" vs "actual" practice

## Round 4: Social, Export, Platform

### Data Export
- Export practice history as CSV
- Generate "practice report" for teachers/coaches
- Share progress as image
- PDF practice log

### Social Sharing
- Share weekly progress as image
- "I'm on a X-day practice streak for guitar"
- Anonymous practice community

### Platform
- iOS app (native)
- macOS app
- Widget: quick-log session

## Round 5: Teacher/Student, AI, Scale

### Teacher / Student
- Teacher creates practice plan for student
- Student logs sessions under teacher's plan
- Teacher sees student's progress
- Practice assignments

### AI Pattern Analysis
- "Your best sessions are on weekday mornings"
- "You practice longer when you use the timer"
- Adaptive session suggestions

### Freemium Polish
- Free: 1 skill, 30-day history
- Track: unlimited logging, 1-year history
- Master: multiple skills, analytics, export, streaks
