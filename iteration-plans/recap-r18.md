# Recap — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Recap widgets across macOS and iOS for quick recap access without opening the app.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: latest recap title
- Medium widget: recap + source + reading time estimate
- Large widget: recap list + generate button
- Interactive widget: generate recap directly
- Widget stack: latest + list

### iOS Home Screen Widgets
- Small: latest recap
- Medium: recap + quick actions
- Large: full recap browser
- Lock screen widget: recap title
- Interactive widget: generate recap

### Daily Digest Widget
- Show today's recap
- Reading time estimate
- Progress: read / unread

### Source Widget
- New articles count per source
- Quick access to source

### Widget Configuration
- In-app widget setup
- Choose which sources to monitor
- Select recap format

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity
