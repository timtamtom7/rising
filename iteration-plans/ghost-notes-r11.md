# Ghost Notes — Round 11: Browser Extension, Email Digest, Import

## Goal
Expand Ghost Notes beyond iOS with browser extension, email digest, and robust import capabilities.

---

## Scope

### Browser Extension
- Save any article from web to Ghost Notes
- "Read later" button in toolbar
- Auto-parse article content (Mercury parser)
- Quick add tags and list from extension popup
- Offline reading sync from extension
- Available for Safari, Chrome, Firefox

### Email Digest
- Weekly "best of" digest email
- "5 articles you saved this week"
- Digest frequency: daily/weekly/monthly (configurable)
- "Still on your list" — articles saved >30 days ago
- Unsubscribe link in every email

### Import Expansion
- Pocket API import (full sync)
- Instapaper API import
- Safari Reading List import (via iCloud)
- Pinboard import
- CSV import for custom sources
- Duplicate detection during import

### Retention Features
- "Are you still reading?" nudges for incomplete articles
- Reading streak: consecutive days with completed articles
- "Saved but never opened" detection and cleanup prompt
- Archive all / archive selected bulk action

---

## Out of Scope
- Social article discovery (R12)
- Audio article narration (R13)
