# Caliber — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Caliber across all major platforms and establish it as the definitive flashcard and spaced repetition tool.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full study experience on Android
- Material You dynamic color support
- Sync with iOS/macOS via cloud
- Google Play Store listing

### Web App (PWA)
- Full-featured web app at caliber.app
- Study from any browser
- Deck management
- Offline support via Service Worker

### Windows / Linux Desktop
- Caliber for Windows/Linux (Electron or Tauri)
- Full desktop app with native notifications
- Windows Store and Linux package distributions

### Apple Vision Pro
- visionOS app
- Spatial study mode: cards floating in space
- 3D progress visualization
- Immersive study environment

### Open Source Core
- Open-source the spaced repetition algorithm and FSRS implementation (caliber-core on GitHub)
- MIT license
- Research community can improve algorithm
- Community-built alternative UIs

### Integration Ecosystem
- Integration registry at caliber.app/integrations
- Notion: create cards from Notion pages
- Obsidian: Obsidian to Caliber sync plugin
- Anki import/export
- Quizlet import
- Zapier/Make/IFTTT: study triggers

### Research & Academic Community
- Publish anonymized learning pattern research (with consent)
- Partner with cognitive science researchers
- Open dataset of spaced repetition performance (anonymized)

---

## Out of Scope
- AI-generated content beyond card suggestions
- Video-based flashcards
- Live tutoring
