# Crisp — Iteration 2 Plan

## What was built (Iteration 1 deep pass)
- Onboarding (4 screens with custom SF Symbol graphics)
- Pricing tiers (Free/Pro/Unlimited)
- Error states (mic denied, transcription failed, storage full, etc.)
- App Store assets (description, keywords, icon concept)

## What's still missing (Iteration 2 focus)

### 1. Organization & Library
- Folder/collection system for notes (Work, Personal, Ideas, etc.)
- Favorite/star important notes
- Sort by: date, duration, title
- Bulk delete
- Search across all transcriptions

### 2. Playback Improvements
- Playback speed control (0.5x, 1x, 1.5x, 2x)
- 15-second skip forward/back buttons
- Audio waveform scrubbing (tap on waveform to seek)
- Background audio playback (continue playing while using other apps)

### 3. Editing & Refining
- Edit transcription text after recording
- Add title manually (default is first few words)
- Merge two recordings into one note
- Split one recording into two

### 4. Sharing & Export
- Share transcription as text
- Share as PDF (formatted with title, date, transcription)
- Export all as ZIP/JSON
- AirDrop support

### 5. Smart Features
- AI summary of each transcription (Apple on-device AI)
- Keyword extraction
- Detect action items (things that sound like tasks)
- "Link to calendar" — if transcription mentions a date/time, offer to create a calendar event

### 6. Widget
- Home screen widget (show last recording, quick record button)
- Small widget: last recording
- Medium widget: last 3 recordings

### 7. Customization
- Custom waveform color
- Recording quality (high/standard)
- Auto-delete after X days (privacy setting)
- Rename recording title

## Custom Graphics for Iteration 2
- Folder/collection icons
- Waveform scrubber design
- Widget mockups
- Share/export menu icons
- Playback speed control UI
