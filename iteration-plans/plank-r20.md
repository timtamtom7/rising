# Plank — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Plank across platforms and establish it as the definitive theme management tool for Mac power users.

---

## Scope

### iOS as Primary (Universal App)
- Plank becomes truly universal: full functionality on iOS and macOS
- iOS can control macOS themes via iCloud
- Unified theme library across devices

### Windows / Linux Desktop
- Plank for Windows/Linux (Electron or Tauri)
- Windows accent color system integration
- Linux: GTK theme support
- Same theme format across platforms

### Web App (PWA)
- Theme marketplace browser at plank.app
- Theme preview from browser
- Account management

### Apple Vision Pro
- visionOS app
- Spatial theme preview
- 3D theme visualization
- Mac-controlled from visionOS

### Open Source Core
- Open-source the theme engine (plank-core on GitHub)
- MIT license
- Community can build alternative UIs and integrations

### Integration Ecosystem
- Integration registry at plank.app/integrations
- Figma: sync themes to Figma variables
- VS Code: theme from Plank in editor
- Wallpaper apps: auto-generate theme from new wallpaper
- Zapier/Make/IFTTT: theme change triggers

### Public Roadmap
- Public roadmap at plank.app/roadmap
- User voting
- Monthly theme featured posts

---

## Out of Scope
- Hardware products
- Game console themes
- Dynamic themes based on music/video
