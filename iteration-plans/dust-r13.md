# Dust — R13: Enterprise & IT Features

## Goal
Position Dust for enterprise IT with MDM support, policy enforcement, compliance reporting, and employee wellness tools.

---

## Scope

### MDM / Managed Device Support
- Mobile Device Management profile for enterprise deployment
- IT can configure default blocklists, allowed apps, focus schedules
- Remote policy push to all enrolled Macs
- Device compliance: block if MDM not enrolled

### Policy Enforcement
- Enforce focus policies on managed devices
- Lock blocking settings so employees can't override
- Configurable: allow personal override (with manager approval) or hard block
- Report violations: when employee bypasses block, log to admin console

### Compliance Reporting
- Focus compliance reports per employee, per team
- Average focus time, focus goal adherence, distraction incidents
- Export for HR / wellness program reporting
- SOC 2 Type II documentation available

### Employee Wellness Dashboard
- Aggregate (anonymized) focus wellness metrics for HR
- "Teams with protected focus time have 15% higher productivity scores"
- Wellness trend analysis
- Recommendations for meeting-free focus blocks

### Focus Break Enforcement
- Enforce mandatory breaks (Pomodoro-style)
- "You've been focused for 50 minutes — take a 10-minute break"
- Break compliance logging

### Volume Licensing
- Apple VPP enrollment for volume distribution
- Enterprise pricing tiers (50+ seats = 30% discount)
- License assignment via MDM

### SSO / SAML
- SAML 2.0 SSO for dashboard access
- Support for Okta, Azure AD, Google Workspace

---

## Out of Scope
- Individual employee performance management
- Real-time surveillance
- Payroll integration based on focus time
