# Coin — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Coin widgets across macOS and iOS for financial awareness without opening the app.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: primary account balance
- Medium widget: balance + spending this month + budget remaining
- Large widget: full financial dashboard — all accounts, spending chart, budget bars
- Interactive widget: add transaction directly from widget
- Widget stack: balance + budget + spending

### iOS Home Screen Widgets
- Small: account balance
- Medium: balance + budget status + recent transaction
- Large: full dashboard
- Lock screen widget: balance
- Interactive widget: add transaction

### Spending Widget
- Weekly/monthly spending chart
- Category breakdown pie chart
- Comparison to last month

### Budget Widget
- Budget progress bars
- Days until budget resets
- "X% of budget remaining"

### Widget Configuration
- In-app widget setup
- Choose which account to show
- Select budget for progress widget

---

## Out of Scope
- Android widgets
- Windows desktop widgets
- Live Activity for financial alerts (future)
