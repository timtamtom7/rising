# Bridge — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Bridge across platforms and establish it as the definitive device connection manager for Apple ecosystems.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Monitor and control devices
- Google Play Store listing

### Windows / Linux Desktop
- Bridge for Windows/Linux (Electron or Tauri)
- Same features as macOS app
- Windows Store listing

### Web App (PWA)
- Remote device management at bridge.app
- View devices from any browser
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial device control
- 3D device visualization
- Mac/device control from visionOS

### Open Source Core
- Open-source the device detection engine (bridge-core on GitHub)
- MIT license
- Community can build integrations for unsupported devices

### Integration Ecosystem
- Integration registry at bridge.app/integrations
- HomeKit full integration
- Sonos, Bose, Sony audio: full control
- Smart home: Philips Hue, Nanoleaf, LIFX
- Zapier/Make/IFTTT: device event triggers

### Public Roadmap
- Public roadmap at bridge.app/roadmap
- User voting on device support
- Monthly featured integrations

---

## Out of Scope
- Direct video streaming from devices
- File transfer protocols
- Network infrastructure management
