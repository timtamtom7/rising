# Muse — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Muse widgets across macOS and iOS for quick project access and playback control.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: current project + play/pause
- Medium widget: project + transport + tempo display
- Large widget: project list + recent + create new
- Interactive widget: play/pause/stop directly
- Widget stack: now playing + project list

### iOS Home Screen Widgets
- Small: now playing project
- Medium: project + controls
- Large: project list + create
- Lock screen widget: now playing
- Interactive widget: transport controls

### Now Playing Widget
- Current project name
- Playback position
- Transport controls

### Recent Projects Widget
- Last 3 projects
- Tap to open

### Widget Configuration
- In-app widget setup
- Choose which projects to show
- Select default view

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity (music playback)
