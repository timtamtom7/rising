# Dreamscape — Round 2: Cloud Sync, Advanced Symbols, Sharing

## Context
R1 built: voice/text dream entry, dream map, symbol detection, dark cosmic UI. R2: cloud sync, deeper symbol analysis, sharing.

## Features

### Cloud Sync (iCloud E2E)
- Enable iCloud sync toggle in settings
- CloudKit container with E2E encryption (user's Apple ID only)
- Sync dreams across devices — conflicts resolved (latest wins)
- Sync status indicator in UI

### Advanced Symbol Analysis
- Symbol clusters — AI groups related symbols ("water + drowning + boat" = fear cluster)
- Symbol diary — each symbol gets a timeline of when it appeared
- Emotional tag per symbol — was this symbol frightening, comforting, confusing?
- Symbol rarity score — common vs. rare symbols in your dreams

### Dream Detail Enhancements
- Mood tag per dream (peaceful, anxious, exhilarating, confusing, dark)
- Lucid dreaming tag — did you know you were dreaming?
- Recurring dream detection — "This is the 4th variation of this dream"
- Attach a photo as dream context (optional)

### Sharing
- Share a dream as a beautiful card (blur names, AI-generates abstract art from symbols)
- Share symbol insight — "I've been dreaming about water for 3 weeks straight"
- Privacy controls — blur names, places, identifying details

### Design Polish
- Star field animation on dream entry screen — subtle parallax
- Dream map — zoom in/out, tap symbol to see details
- Symbol network gets smoother spring animations
