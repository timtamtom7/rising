# Notch — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Notch across platforms and establish it as the definitive menu bar management tool.

---

## Scope

### Windows / Linux Desktop
- Notch for Windows (Electron or Tauri)
- System tray management for Windows
- Taskbar management for Linux
- Same preset system across platforms

### Web App (PWA)
- Configuration at notch.app
- Preset management from browser
- Analytics dashboard
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial menu bar configuration
- 3D preset visualization
- Mac control from visionOS

### Open Source Core
- Open-source the menu bar control engine (notch-core on GitHub)
- MIT license
- Community can build alternative UIs

### Integration Ecosystem
- Integration registry at notch.app/integrations
- Popular menu bar apps: add Notch control
- Time tracking: link to menu bar status
- Zapier/Make/IFTTT: preset change triggers

### Public Roadmap
- Public roadmap at notch.app/roadmap
- User voting on features
- Monthly blog posts

---

## Out of Scope
- Android (no system tray equivalent)
- Full desktop environment management
- Game console support
