# Cast — R9: Post-Launch, Monetization Refinement, Roadmap

## Goal
Analyze launch metrics, refine business model, and plan the next feature cycle.

---

## Scope

### Launch Metrics
- Downloads from App Store Connect
- Revenue
- Retention: day 1/7/30
- Crash rate vs beta
- IAP conversion rate
- Average cast session length
- Top used features

### Business Model Refinement
- Review Pro pricing: too high/low?
- Consider lifetime license
- Family sharing for Pro
- Educational discount
- Referral program

### Feature Roadmap (Next Cycle)
From user reviews and analytics:
1. **Overlay mode**: cast with webcam overlay (for gaming streams)
2. **YouTube/Twitch streaming**: directly stream to YouTube Live or Twitch
3. **Apple Watch remote**: control cast from Apple Watch
4. **Sound mode**: cast audio only to multiple speakers (multi-room audio)
5. **Presentation mode**: teleprompter text overlay

### Maintenance
- macOS update compatibility within 1 week of beta
- Xcode update within 1 week
- Security patches within 48 hours
- Dependency updates

### Community
- GitHub discussions
- Changelog blog
- Social media updates

---

- End-of-life planning
