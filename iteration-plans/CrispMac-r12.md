# CrispMac R12 — Sharing

## Theme
Enable team-level voice note sharing, collaborative updates, and seamless conversion of voice memos into actionable tasks.

## Features
- **Voice Note Sharing with Team** — Share individual notes or entire threads with team members; recipients can play, comment at specific timestamps, and react; access control: viewer or editor
- **Voice-Based Team Updates** — Record a voice update, tag it as "Team Update"; team members receive a notification; the update is automatically summarized and posted to a linked Slack/Discord channel (optional integration)
- **Speech-to-Task Pipeline** — Convert any voice note to a task card (title, description, due date, assignee); bulk convert: select multiple notes → AI batches them into task cards in one pass; integrates with Linear, Notion, or native Reminders
- **Team Voice Digest** — Weekly AI-generated digest of team voice notes: top themes, action items extracted, unresolved questions surfaced — delivered as a readable summary + action list

## Technical Notes
- **SharingService:** Secure signed URLs for shared notes; timestamp-level commenting; reaction emoji stored per user-note pair
- **TeamUpdateIntegration:** Webhook-based outbound integration to Slack/Discord; summary generated by local LLM from full transcript; configurable per-channel
- **TaskConverterPipeline:** Batch-processing endpoint for speech-to-task conversion; outputs structured JSON matching Linear/Notion/Reminders schemas; retry with exponential backoff on API errors
- **DigestGenerator:** Aggregates team notes from the past 7 days; topic modeling to surface top themes; action item extraction across all notes; rendered as HTML email + in-app card
