# Coin — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Coin to Android and web to become the definitive privacy-first personal finance app across all platforms.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full financial management
- Material You dynamic color
- Manual transaction entry + receipt scan
- Google Play Store listing

### Web App (PWA)
- Full-featured web app at coin.app
- View accounts, transactions, budgets
- Add transactions
- Offline support via Service Worker

### Windows / Linux Desktop
- Coin for Windows/Linux (Electron or Tauri)
- Same features as macOS app
- Windows Store and Linux packages

### Apple Vision Pro
- visionOS app
- Spatial financial dashboard
- 3D spending visualization
- Privacy-preserving: financial data only on device

### Open Source Core
- Open-source the financial engine (coin-core on GitHub)
- MIT license
- Community can build alternative UIs
- Researchers can study personal finance patterns

### Integration Ecosystem
- Integration registry at coin.app/integrations
- Receipt scanning: integrate with external OCR if user opts in
- Tax software: TurboTax, H&R Block import
- Zapier/Make/IFTTT: financial triggers
- Bank CSV export → Coin import (privacy-preserving approach)

### Public Roadmap
- Public roadmap at coin.app/roadmap
- User voting on features
- Monthly finance tips blog

---

## Out of Scope
- Bank account connections (privacy choice)
- Investment tracking
- Cryptocurrency tracking
- Automated bill pay
