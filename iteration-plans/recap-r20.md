# Recap — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Recap across platforms and establish it as the definitive content curation and summarization tool.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full recap reading and generation
- Google Play Store listing

### Windows / Linux Desktop
- Recap for Windows/Linux (Electron or Tauri)
- Same features as macOS app
- Windows Store and Linux packages

### Web App (PWA)
- Recap access at recap.app
- Read and generate recaps from browser
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial recap reading
- 3D content organization
- Immersive reading environment

### Open Source Core
- Open-source the summarization engine (recap-core on GitHub)
- MIT license
- Researchers can improve ML models

### Integration Ecosystem
- Integration registry at recap.app/integrations
- RSS readers: Reeder, NetNewsWire import
- Note apps: Notion, Obsidian link
- Slack: daily digest to channel
- Zapier/Make/IFTTT: recap triggers

### Public Roadmap
- Public roadmap at recap.app/roadmap
- User voting on features
- Monthly content digest blog

---

## Out of Scope
- Full social media aggregation
- Video/audio content summarization
- Advertising platforms
