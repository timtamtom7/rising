# Zones — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Zones widgets across macOS and iOS for quick zone management without opening the app.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: current zone name
- Medium widget: current zone + 3 quick switch buttons
- Large widget: full zone dashboard + all zones + recent history
- Interactive widget: switch zone directly from widget
- Widget stack: current + quick switch

### iOS Home Screen Widgets
- Small: current zone
- Medium: zone + quick switch
- Large: full dashboard
- Lock screen widget: current zone
- Interactive widget: switch zone

### Zone History Widget
- Last 5 zone changes
- Time spent in each zone today

### Household Widget (Home tier)
- Family members' current zones
- "Everyone is in Home Zone"

### Widget Configuration
- In-app widget setup
- Choose which zones to show
- Select default view

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity
