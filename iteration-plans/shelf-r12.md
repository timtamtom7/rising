# Shelf — R12: Collaboration & Team Features

## Goal
Enable Shelf to be shared across a team for collaborative project management, shared resources, and team activity awareness.

---

## Scope

### Shared Shelves
- Create a team shelf and share via iCloud
- Team members can contribute items to shared shelf
- Items tagged with contributor name and timestamp
- Shared shelves for: "Team Resources", "Onboarding Docs", "Shared Project Assets"

### Team Activity Feed
- See what teammates have been working on (with permission)
- Activity stream: "Tommaso opened Budget Q4.xlsx", "Sara added new design mockup"
- Privacy setting per user: opt-in only, can hide specific items
- No keystroke logging — only explicitly shared shelf interactions

### Collaborative Stacks
- Create a stack collaboratively — invite teammates to contribute
- Version history for shared stacks
- Comment on shared items (threaded comments)
- @mention teammates to draw attention to an item

### Team Admin Controls
- Shelf admin can manage team members (invite, remove, set roles)
- Roles: Admin, Editor, Viewer
- Admin can pin items to team shelf
- Admin can set content policies (what can be shared)

### Guest Access
- Invite external collaborators (clients, contractors) to specific shared shelves
- Guest gets read-only or contributor access per shelf
- External guests don't need an account — link-based access with email verification

### iCloud Sharing Improvements
- Share entire shelf via iCloud sharing link
- Real-time sync of shared shelf changes
- Commenting and reactions sync instantly

---

## Out of Scope
- Real-time co-editing of documents (use dedicated apps)
- Video conferencing
- Task/issue tracking integration
