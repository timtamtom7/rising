# Notch — R18: Advanced Widgets & Desktop Integration

## Goal
Expand Notch widgets across macOS and iOS for quick menu bar control without opening the app.

---

## Scope

### macOS Desktop Widgets (WidgetKit)
- Small widget: current preset name
- Medium widget: preset + quick toggles for 3 items
- Large widget: full config + presets + analytics
- Interactive widget: toggle items directly from widget
- Widget stack: preset + toggles

### iOS Home Screen Widgets
- Small: current preset
- Medium: preset + toggles
- Large: full dashboard
- Lock screen widget: preset name
- Interactive widget: toggle items

### Quick Toggle Widget
- Dedicated widget for 1-tap item toggle
- Show/hide specific items directly

### Preset Widget
- List of presets
- Tap to apply directly

### Widget Configuration
- In-app widget setup
- Choose which items to show toggles for
- Select default preset

---

## Out of Scope
- Android widgets
- Windows widgets
- Live Activity for menu bar
