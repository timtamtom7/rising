# Ghost Notes — Rounds 3, 4, 5

## Round 3: Search, Polish, Retention

### Advanced Search
- Full-text search across all saves
- Filter by: read/unread, domain, list, date range
- Search within a specific list
- Keyboard shortcut: Cmd+K for search

### Retention Features
- "You haven't visited in X days" — re-surface old ghosts
- "Did you mean to read this?" on old saves
- Automatic cull suggestion: "You haven't opened this in 30 days — cull?"

### Browser Extension Polish
- Right-click context menu: "Save to Ghost Notes"
- Auto-detect article title and favicon
- Visual confirmation when saved
- Extension popup showing recent saves

### Bookmarklet Polish
- Drag-to-bookmarks bar
- Visual confirmation animation
- Works on mobile Safari

---

## Round 4: Email, Automation, Social

### Email Integration
- Weekly digest email (resend)
- "Your Ghost Notes are here" — configurable frequency
- Email summary with top 5 unread
- One-click read/cull from email

### Auto-Import
- Import from Pocket (Pocket API integration)
- Import from Instapaper
- Import from browser bookmarks (CSV export)

### Share / Export
- Share a saved article (send link to someone)
- "Ghost Note" — share a quote/highlight from an article
- Public profile (optional): "tomas.so/ghost/username"

### Teams Tier
- Invite collaborator by email
- Shared lists
- Role: admin/member

---

## Round 5: Platform, API, Scale

### iOS Share Extension
- iOS share sheet to save to Ghost Notes
- Save from Safari, News, any app
- Background save (don't leave current app)

### Android App
- Native Android app
- Save from Chrome browser share

### API for Developers
- Public API: save URL, get saves, mark read/cull
- Zapier integration
- Buffer integration (save articles from social media)

### Pro Features
- Offline reading (save article for offline)
- Readwise integration (export highlights to Readwise)
- API access
