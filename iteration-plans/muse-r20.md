# Muse — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Muse across platforms and establish it as the definitive AI-assisted music creation tool.

---

## Scope

### iOS as Full Platform
- Muse becomes a full-featured iOS music app
- Interop with macOS via iCloud
- Touch instruments for iOS

### Android App
- Native Android app (Kotlin)
- Full music creation on Android
- Google Play Store listing

### Windows / Linux Desktop
- Muse for Windows/Linux (Electron or Tauri)
- Same features as macOS app
- VST3 support on Windows
- Windows Store and Linux packages

### Web App (PWA)
- Project playback at muse.app
- Collaborator access for commenting
- Sample library browsing
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial mixer view
- Immersive music composition
- 3D audio visualization

### Open Source Core
- Open-source the MIDI engine and audio engine (muse-core on GitHub)
- MIT license
- Community can build instruments and effects

### Integration Ecosystem
- Integration registry at muse.app/integrations
- Audio interface support
- MIDI controller support
- GarageBand / Logic import
- YouTube, SoundCloud export
- Zapier/Make/IFTTT: project triggers

### Public Roadmap
- Public roadmap at muse.app/roadmap
- User voting on features
- Monthly music featured posts

---

## Out of Scope
- Distribution to Spotify/Apple Music
- Royalty accounting
- Music licensing
