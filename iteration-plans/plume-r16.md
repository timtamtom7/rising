# plume — Round 16: Plume 2.0, Audiobooks, Reading Experience Redesign

## Goal
Ship Plume 2.0 — a comprehensive redesign that makes Plume the definitive reading companion — now including audiobooks and a fully reimagined reading experience.

---

## Scope

### Plume 2.0 — Major Redesign
- **New UI**: Editorial, magazine-inspired design — beautiful typography, generous whitespace
- **Book cover art**: Full-bleed book covers as backgrounds — immersive, beautiful
- **Reading mode redesign**: Distraction-free reading tracking — current page, time remaining, session stats
- **SwiftUI 5 + iOS 18+**: Full redesign with latest SwiftUI capabilities, interactive widgets, App Intents for Siri

### Audiobook Support
- **Audiobook library**: Track audiobooks alongside print and e-books
- **Apple Books audiobook sync**: If listening via Apple Books, sync progress automatically
- **Audible import**: Import audiobooks from Audible (via Amazon data export if available)
- **Audiobook-specific stats**: Listening speed, chapters completed, narrator tracking
- **"Read vs. Heard" stats**: Combined reading stats across print, e-book, and audiobook

### Apple Intelligence Integration
- **Siri + Plume**: "Hey Siri, log that I read 20 pages" — one voice command
- **Reading suggestions**: Apple Intelligence suggests reading time — "You usually read at 9 PM — want to start?"
- **Live Activities**: Active reading session on Lock Screen — current page, time today

### Reading Experience
- **Page-turn animation**: Elegant page-turn animation when logging progress
- **Chapter milestones**: Celebrate finishing chapters
- **Book completion**: Full celebration animation when finishing a book
- **Series tracking**: When you finish a book that's part of a series, prompt to add the next
