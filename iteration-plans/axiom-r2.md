# Axiom — Round 2: Belief Network, AI Deep Dive, Evidence Expansion

## Context
R1 built: belief entry, evidence collection, AI stress test, belief score, dark charcoal UI. R2: deeper analysis, belief network, AI conversation.

## Features

### Belief Network
- Beliefs are connected — "I'm not creative" links to "Creative work is risky"
- Visual network graph — tap a belief, see what other beliefs it connects to
- Core beliefs vs. surface beliefs — AI identifies which beliefs are foundational
- Belief chains — "This belief stems from this childhood experience"

### AI Deep Dive Conversation
- Instead of static questions — a real back-and-forth dialogue with AI about a belief
- AI asks: "What's the evidence for this? What would have to be true for this to be false?"
- User can ask AI questions back: "Is this belief common?"
- Record the conversation — it becomes evidence

### Evidence Expansion
- Link a belief to a specific memory (voice note or photo)
- Link to an external resource (article, book excerpt) that supports or contradicts
- Evidence confidence rating — how certain are you about this evidence?
- AI suggests: "Have you considered this opposing viewpoint?"

### Belief Outcome Tracking
- Check back in 30/60/90 days — has the belief survived or shifted?
- Before/after score comparison
- Belief obituary — "This belief is no longer held. Here's why."

### Design Polish
- Evidence cards — crisp red (contradicting) and green (supporting) borders
- Network graph — smooth force-directed, tap to expand
- Debate animations — arguments slide in, counter-arguments counter
