# Margin — Round 2: AI Insights, Location, Social

## Context
R1 built: moment capture, micro-reflection, pattern analysis, paper-like design. R2: deeper AI, location context, anonymous social.

## Features

### AI Pattern Deep Dive
- "Deep thought" detection — AI identifies when you had a genuinely novel thought (vs. routine worrying)
- Mood tag per moment — anxious, curious, creative, melancholy, excited
- Thread detection — "This thought was still on your mind 3 moments later"
- Weekly AI summary: "Most of your dead time this week was spent planning. You thought about the same project 5 separate times."

### Location Context (Privacy-First)
- If location permission given — tag moments with where you were (work, home, transit, waiting)
- If denied — time-based patterns only (same insights, no location)
- "Where do you think best?" — AI surfaces location patterns
- Location data never leaves device, never synced

### Moment Threads
- Link related moments — "this is a continuation of that thought"
- Thread view — follow a thought across the whole day
- Abandoned thread reminder — "You started thinking about this 3 days ago"

### Anonymous Social
- Share a moment anonymously to a public feed
- See what others in similar situations (waiting for the same thing) are thinking
- "Most common thoughts at traffic lights" — aggregate insights
- No names, no accounts needed

### Design Polish
- Moment cards — slight paper texture, torn-edge effect
- Capture button — gentle pulse animation
- Pattern cards — illustrated with simple line drawings
- Digest card — feels like a handwritten note
