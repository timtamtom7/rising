# Nimbus — R15: iOS Companion App

## Goal
Ship Nimbus as a native iOS and iPadOS app for note-taking on the go, with full editing and collaboration.

---

## Scope

### iOS App
- Full-featured note-taking on iPhone and iPad
- Rich text editor with all formatting options
- Notebook and folder management
- Quick note capture from widget
- Apple Pencil support for handwriting

### iPad Optimization
- Full iPad layout with sidebar + editor split
- Trackpad/mouse support with precise cursor
- Keyboard shortcuts matching macOS app
- Stage Manager / Split View support
- External display for presentations

### Apple Watch App
- Quick note dictation: record voice note → converted to text in Nimbus
- Glance: recent notes list
- Complication: "New Note" quick action
- Haptic feedback for note saved

### iOS Widgets
- Small widget: most recent note
- Medium widget: 3 recent notes + quick add button
- Large widget: notebook list + recent notes
- Lock screen widget: recent note title
- Interactive widget: create new note directly

### Siri & Shortcuts
- "Take a note" → opens quick note with voice
- "Create note titled X with content Y"
- "Get my meeting notes from today"
- Shortcuts actions: create note, search notes, list notebooks

### Collaboration on iOS
- Real-time collaborative editing
- Commenting and mentions
- Presence indicators

---

## Out of Scope
- Android companion (separate round)
- Full wiki management on watch
