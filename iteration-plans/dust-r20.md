# Dust — R20: Platform Expansion & Long-Term Vision

## Goal
Expand Dust to all major platforms and establish it as the definitive focus and productivity companion.

---

## Scope

### Android App
- Native Android app (Kotlin)
- Full focus management, blocking, statistics
- Digital Wellbeing integration (Android's built-in focus features)
- Google Play Store listing

### Windows / Linux Desktop
- Dust for Windows/Linux (Electron or Tauri)
- Same blocking engine and ML features
- Windows Store and Linux package distributions

### Web App (PWA)
- Focus dashboard at dust.app
- View stats and history from browser
- Basic focus session start/stop
- Offline support via Service Worker

### Apple Vision Pro
- visionOS app
- Spatial focus timer floating in workspace
- Ambient focus mode visualization
- 3D productivity charts

### Open Source Core
- Open-source the blocking engine (dust-core on GitHub)
- MIT license
- Community can build alternative UIs
- Research use of anonymized focus patterns

### Integration Ecosystem
- Integration registry at dust.app/integrations
- Slack: "Focus mode on" status sync
- Zoom/Teams: auto-decline meetings during focus
- Notion, Asana, Linear: task focus mode
- Zapier/Make/IFTTT: focus triggers

### Public Roadmap
- Public roadmap at dust.app/roadmap
- User voting on features
- Monthly productivity tips blog

---

## Out of Scope
- Payment/gamification tied to focus time
- Investment/financial features
- Social media presence tracking beyond blocking
