# Muse — R15: iOS Companion App

## Goal
Ship Muse as a native iOS and iPadOS app for music creation, recording, and collaboration on the go.

---

## Scope

### iOS App
- Full music creation on iPhone and iPad
- MIDI recording and editing
- Audio recording (iPhone mic)
- Collaboration playback and commenting
- Project management

### iPad Optimization
- Full iPad layout with track view
- External MIDI controller support
- Split View / Stage Manager
- Keyboard shortcuts matching macOS
- External display for mixer view

### Apple Watch App
- Watch: metronome with haptic tick
- Quick tempo adjustment
- Record trigger from watch
- Haptic count-in

### iOS Widgets
- Small widget: current project name
- Medium widget: project + playback controls
- Large widget: project list + create new
- Lock screen widget: now playing
- Interactive widget: play/pause from widget

### External MIDI
- Connect MIDI controllers via Camera Connection Kit
- Core MIDI support for all MIDI devices
- Bluetooth MIDI

### Logic Remote Features
- Use iPhone/iPad as a remote control for Muse on Mac
- Mixer view, transport controls
- Touch instruments

---

## Out of Scope
- Full DAW on watch
- Audio mastering on iOS
