# Cast — R12: Collaboration & Shared Watch Parties

## Goal
Enable Cast for shared viewing experiences with watch parties, shared queues, and social features.

---

## Scope

### Watch Party
- Synchronized watching with friends (remote)
- Watch same stream together with chat
- Watch Party host controls playback for all
- "Tommaso paused the stream" — everyone pauses
- Sync accuracy: within 500ms across devices

### Shared Queue
- Create shared watch queue with friends
- Anyone can add to queue
- Vote on next item
- Queue visible to all party members

### Social Features
- Follow other Cast users
- See what friends are casting
- Activity feed: "Sara just started watching..."
- Share streams to friends

### Room Casting
- Cast to a named room (Living Room, Bedroom)
- Anyone on the same network can join the room
- Shared playback control
- Room chat

### Public Watch Parties
- Host public watch parties (streaming content you have rights to)
- Attendees join via link
- Host moderates

---

## Out of Scope
- Pirated content streaming
- Video call integration
- Subscription management for attendees
