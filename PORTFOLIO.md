## Quality Standards

Every app in this portfolio meets the following baseline — non-negotiable.

### Design System
- Unified color system shared across all 44 apps
- Consistent typography scale using SF Pro
- Shared component library in Swift Package Manager
- Dark mode and light mode first-class citizens

### Accessibility
- Full VoiceOver support — every screen tested
- Dynamic Type support across all text
- Minimum contrast ratios meeting WCAG AA
- Reduce Motion respected throughout

### iCloud Sync
- CloudKit integration with conflict resolution
- End-to-end encryption for sensitive data
- Seamless handoff between devices
- Privacy-preserving architecture by default

### Widgets
- WidgetKit implementation on all iOS apps
- Small, medium, and large widget sizes
- Interactive widgets where context allows
- Lock screen widgets on supported devices

### Shortcuts
- Siri Shortcuts support with deep links
- URL schemes for all apps
- Custom intents for complex automations
- Automation-friendly architecture

### Localization
- All user-facing strings in Localizable.strings
- English as primary, structured for localization
- Date, time, and number formatters using locale-aware APIs
- Right-to-left layout support

## Design Language

### Platform: iOS 26 / macOS 15

Full native adoption of Apple's latest frameworks. No legacy UIKit where SwiftUI suffices. All apps are SwiftUI-native unless a specific capability requires UIKit.

### Architecture: MVVM + Services

```
View ←→ ViewModel ←→ Services ←→ Repository
                ↓
           Models (Codable, Equatable)
```

- **Views:** Pure SwiftUI, no business logic
- **ViewModels:** @Observable, no side effects
- **Services:** Single-responsibility, protocol-based
- **Repositories:** Data access abstraction

### SwiftUI Native
- Native gestures and animations
- NavigationStack for all navigation
- @Observable for state management
- Structured concurrency (async/await)

### Dark + Light Mode
- Full semantic color tokens
- Automatic adaptation via Color assets
- Premium feel in both modes — not an afterthought
- Automatic accent color matching system

### Dynamic Type
- All text scales from xSmall to AX5
- Layouts adapt without truncation
- Minimum touch targets preserved
- Priority-based layout priorities

## Technical Philosophy

**Privacy by default.** On-device ML, local-first data, minimal network calls. Users trust us with intimate data; we honor that.

**Craft over velocity.** Every pixel, transition, and interaction is intentional. Speed matters, but not at the cost of feel.

**Shared infrastructure.** One design system, one component library, one architecture pattern. Building 44 apps means building leverage.

**Ship and iterate.** Each app is a complete product. shipped, then refined based on real use.
